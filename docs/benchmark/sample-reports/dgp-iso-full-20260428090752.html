Conclusions

No delta storage win detected. Prometheus shows zero delta encode events and no increase in deltaglider_delta_bytes_saved_total during compression phases. Typical causes: ISO contents did not beat max_delta_ratio (passthrough fallback), or the running proxy build predates .iso eligibility in FileRouter.

PUT throughput drops from 125.6 MB/s (passthrough) to 6.9 MB/s (compression) — delta encode is CPU-heavy vs streaming writes.

The encrypted backend bucket measured 104.3 MB/s PUT vs passthrough — AES-GCM adds measurable CPU cost while preserving logical bytes on read.
